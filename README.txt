No-AFK-Minecraft will keep your character moving around on the spot to avoid AFK'ing on minecraft servers!

Dependencies:
Pyautogui -> 'python -m pip install pyautogui'

Run this simple command into your terminal/cmd and click app.py to get going!
The program will prompt you to enter anything before it starts, this can be as simple as pressing enter.
From then you will have 5 seconds to wait.
The program will keep running until you press q.
